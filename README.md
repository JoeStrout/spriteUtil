# spriteUtil

In this repo, we're developing a set of general-purpose sprite utilities for [Mini Micro](https://miniscript.org/MiniMicro).

Once we're happy with it, it will get rolled into the [Mini Micro /sys disk](https://github.com/JoeStrout/minimicro-sysdisk).
